# shawn-weekly
## 周刊说明
- **周刊内容**: 对一周内阅读的资讯、技术内容精品（个人向）进行总结，分类大致包含“业界资讯”、“技术博客”、“开源项目”和“工具分享”等。为减少阅读负担提高记忆留存率，每类下内容数一般不超过3条。<br>
- **更新时间**: 周一<br>
