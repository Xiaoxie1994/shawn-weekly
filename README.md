# shawn-weekly
## 声明
- **周刊内容**: 对一周内阅读的资讯或技术内容精品（个人向）进行总结，分类大致包含“业界资讯”、“技术博客”、“开源项目”和“工具分享”等。为减少阅读负担提高记忆留存率，每类下内容数一般不超过3条。<br>
- **更新时间**: 星期天<br>

## 订阅方式
避免错过每期内容，可以通过以下方式订阅：
- 个人博客：[肖恩技术周刊](https://www.shawnxie.top/categories/tf-weekly), [RSS订阅](https://www.shawnxie.top/feed.xml)
- 公众号：肖恩杂谈
